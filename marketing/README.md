# Marketing Materials

This directory contains all marketing content for the OctaveHQ clone announcement.

## LinkedIn Content Package

All LinkedIn-related materials are organized in `/linkedin`:

### `/linkedin/posts`
Four ready-to-post LinkedIn content variations:
- `linkedin_post.md` - Full-length detailed post (800-1000 words)
- `linkedin_post_short.md` - Concise version (300-400 words)
- `linkedin_post_story.md` - Narrative-driven post with storytelling
- `linkedin_post_viral.md` - Viral-optimized version with engagement hooks

### `/linkedin/guides`
Comprehensive guides and instructions:
- `START_HERE.md` - Quick start guide for choosing and posting
- `WHICH_POST_TO_USE.md` - Decision guide for selecting the right post variant
- `LINKEDIN_POST_GUIDE.md` - Detailed posting instructions and tips
- `VISUAL_ASSETS_GUIDE.md` - Guidelines for screenshots and visuals
- `README_LINKEDIN_POST.md` - Complete LinkedIn package documentation
- `POST_SUMMARY.md` - Overview of all post variants
