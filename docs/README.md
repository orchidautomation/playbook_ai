# Documentation

This directory contains all project documentation organized by category.

## Structure

### `/architecture`
Core technical documentation and design decisions:
- `ARCHITECTURE.md` - System architecture and design patterns
- `IMPLEMENTATION_PLAN.md` - Detailed implementation roadmap
- `AGNO_COMPLIANCE_AUDIT.md` - Agno framework compliance review

### `/research`
Research and analysis documents:
- `OCTAVE_RESEARCH_REPORT.md` - Research on OctaveHQ platform
- `OCTAVE_SAMPLE_BRIEF.md` - Sample sales brief structure

### `/phases`
Phase completion summaries tracking project milestones:
- `PHASE4_COMPLETION_SUMMARY.md` - Phase 4: Sales playbook generation
- `PHASE4.5_PRE_COMPLETION_SUMMARY.md` - Pre-completion review
- `PHASE5_PRE_COMPLETION_SUMMARY.md` - Phase 5 planning

### `/simplification`
Code simplification and optimization documentation:
- `SIMPLIFICATION_COMPLETE.md` - V1 simplification summary
- `SIMPLIFICATION_EXAMPLES.md` - Before/after code examples
- `SIMPLIFICATION_V2.md` - Future optimization approach

### `/examples`
Example code and guides:
- `parallel-outputs/` - Agno parallel workflow examples and guides
